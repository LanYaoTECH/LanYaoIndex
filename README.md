# 澜鳐生物科技官网

## 项目介绍
这是澜鳐生物科技（LanYaoBio）的官网，基于GitHub Pages搭建。

## 快速开始
[访问澜鳐生物科技官网](https://lanyaotech.github.io/LanYaoIndex/)https://lanyaotech.github.io/LanYaoIndex/

## 功能特性
- 响应式设计，适配移动端和桌面端
- 平滑滚动和动画效果
- 产品服务标签页切换
- 联系表单提交功能
- 多语言切换支持

## 项目维护
- 更新内容：修改对应的HTML、CSS或JavaScript文件
- 添加新页面：创建新的HTML文件并在导航中添加链接
- 更新资源：替换images目录下的相应文件
