# 澜鳐生物科技官网

## 项目介绍
这是澜鳐生物科技（LanYaoBio）的官方网站，基于大橡科技网站风格设计。网站包含公司简介、核心技术、产品服务、新闻动态和联系方式等板块。

## 项目结构
```
├── index.html          # 网站首页
├── src/
│   ├── css/            # 样式文件
│   │   └── style.css   # 主样式文件
│   ├── js/             # JavaScript文件
│   │   └── main.js     # 主JavaScript文件
│   └── images/         # 图片资源
│       ├── logo.svg    # 公司Logo
│       ├── hero-bg.svg # 英雄区域背景
│       └── *.jpg       # 其他图片资源
├── package.json        # 项目配置文件
└── .gitignore          # Git忽略文件
```

## 技术栈
- HTML5
- CSS3 (响应式设计)
- JavaScript (原生)
- SVG (矢量图形)

## 快速开始
1. 克隆项目：`git clone https://github.com/your-username/LanYaoIndex.git`
2. 安装依赖：`npm install`
3. 启动服务：`npm start`
4. 打开浏览器访问：`http://localhost:3000`

## 功能特性
- 响应式设计，适配移动端和桌面端
- 平滑滚动和动画效果
- 产品服务标签页切换
- 联系表单提交功能
- 多语言切换支持

## 项目维护
- 更新内容：修改对应的HTML、CSS或JavaScript文件
- 添加新页面：创建新的HTML文件并在导航中添加链接
- 更新资源：替换images目录下的相应文件
